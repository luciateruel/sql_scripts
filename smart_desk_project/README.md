# Smart Desk Project - SQL Practice

This project showcases SQL skills for **data engineering**. It includes queries for handling and analyzing structured datasets, focusing on sales, profit, and account performance.

## Features
- Aggregate and analyze sales and profit
- Compare performance across regions and industries
- Identify top-performing categories and accounts
- Practice joins, unions, window functions, and CTEs

## Structure
- `raw_data/`: example datasets
- `sql/`: SQL scripts

> Goal: demonstrate SQL proficiency and a data engineering mindset in real-world business datasets.
